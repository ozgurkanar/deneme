# Burakin Uzun Yolu (TR)

Burakin Uzun Yolu, **Türkçe hikâyeli**, tek dokunuşla oynanabilen, Android için hazırlanmış 2D platform macerasıdır.

## Öne çıkanlar
- Tamamen **Türkçe arayüz ve hikâye**
- Burak'ın babasına ulaşma yolculuğu
- Daha **belirgin ve opak platform çizimleri** (şeffaf platform problemi giderildi)
- Bölüm bölüm ilerleyen daha dikkatli level tasarımı
- Kodla üretilen görseller, arka planlar ve sesler
- Dokunmatik kontrol + klavye desteği
- Checkpoint, moving platform, köprü anahtarı, düşman, koleksiyon öğeleri

## Kontroller
### Telefonda
- Sol alt: sola / sağa hareket
- Sağ alt: zıplama
- Sağ alt yıldız: etkileşim / köprü açma
- Sağ üst: duraklat

### Test için klavye
- A / Sol ok: sola git
- D / Sağ ok: sağa git
- W / Yukarı / Space: zıpla
- E / Enter: etkileşim
- P: duraklat
